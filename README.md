# ⚔️ BS-Parody

> Мультиплеерный 3D-шутер, вдохновлённый **Brawl Stars**, с командными боями в реальном времени на карте **Retropolis**.

---

## 📖 Описание

**BS-Parody** — это онлайн-игра с видом сверху, в которой две команды (**🟢 Green** и **🔴 Red**) сражаются друг с другом на одной арене. Игроки могут переключаться между дальними и ближними атаками, автоматически захватывать цели в радиусе действия, создавать игровые комнаты или находить случайных соперников, и всё это — с синхронизацией через **Photon PUN 2**.

---

## ✨ Основные возможности

| Функция | Описание |
|---|---|
| 🌐 **Мультиплеер** | Онлайн до 10–20 игроков через Photon PUN 2 |
| ⚔️ **Два режима атаки** | Переключение между ближним боем (melee) и дальним боем (ranged) |
| 🎯 **Автонаведение** | Автоматический выбор ближайшей цели с возможностью ручного переключения |
| 🏟️ **Лобби-система** | Создание комнат, список доступных комнат, автоматический подбор (PvP) |
| 👥 **Командная игра** | Разделение на две команды (Green / Red) с подсчётом очков |
| ❤️ **Система здоровья** | HP-бар с цветовой индикацией команды, получение урона, смерть и респаун |
| ⏱️ **Таймер матча** | Ограниченное время боя, определение победителя по окончании |
| 📊 **Статистика** | Отслеживание побед, поражений и личных показателей игрока |
| 🎮 **Input System** | Современная система ввода Unity с поддержкой клавиатуры и геймпада |
| 🎨 **URP** | Universal Render Pipeline для качественной графики |

## 🎮 Игровой процесс

### Лобби
1. Игрок подключается к **Photon Master Server** и попадает в лобби  
2. Можно **создать комнату**, **присоединиться к существующей** или нажать **«В бой»** для автоматического подбора  
3. Мастер комнаты (хост) запускает игру; остальные ждут

### Бой
1. Игроки распределяются по командам (**Green** / **Red**)  
2. Управление персонажем: **WASD** — движение, **Shift** — бег  
3. При остановке автоматически определяется ближайший враг  
4. Клавиши **1/2** — переключение между ближним и дальним боем  
5. **ЛКМ (Fire)** — атака по текущей цели  
6. Снаряды (CloudBall) летят к цели и наносят урон  
7. При смерти — очко противоположной команде, игрок покидает бой  
8. По истечении таймера — побеждает команда с большим количеством очков

## 🚀 Начало работы

### Требования
- **Unity Hub** с установленным Unity **2021.3.30f1**
- Аккаунт на [Photon Engine](https://www.photonengine.com/) (бесплатный план)

### Установка

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/<your-username>/BS-parody.git

# 2. Откройте проект в Unity Hub
#    Add → выберите папку BS-parody

# 3. Настройте Photon App ID
#    Window → Photon Unity Networking → PUN Wizard → вставьте свой App ID

# 4. Откройте сцену NewGameLobby и нажмите Play
```

## 📜 Ключевые скрипты

| Скрипт | Описание |
|---|---|
| `ConnectionToServer.cs` | Подключение к Photon, управление лобби и комнатами |
| `PlayerController.cs` | Движение, вращение и анимации персонажа |
| `Aim.cs` | Автонаведение, выбор цели, дальняя/ближняя атака |
| `Bullet.cs` | Логика снарядов (движение, столкновение, урон) |
| `PlayerSetting.cs` | HP, получение урона, смерть, события победы/поражения |
| `GameManager.cs` | Таймер матча, подсчёт очков, определение победителя |
| `SwitchAttack.cs` | Переключение между режимами атаки (melee / ranged) |
| `CameraFollow.cs` | Следование камеры за игроком |
| `GameEventsManager.cs` | Менеджер игровых событий (победа, поражение) |
